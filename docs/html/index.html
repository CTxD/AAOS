Setup

Follow these steps to do the initial setup of the project

Docker

Installation of tools

Install the following tools:

Docker: https://docs.docker.com/install/

Docker-Compose: https://docs.docker.com/v17.09/compose/install/

Project Installation

Do the following to setup the project

  1. Clone git reposiroty
    • https://github.com/CTxD/FreeTACOS
  2. Start the Docker containers
    • $ make start                              # From root of project
  3. Clone circle into FreeTacos project
    • $ make enter                              # Enter the Docker container
$ cd aarch/circle-stdlib                  # Enter the circle dependency folder
$ git submodule update --init --recursive # Clone the circle repository
$ ./configure -p aarch64-elf- -r 4        # Configure circle for Raspberry Pi 4
$ make                                    # Build Circle
  4. Generate arinc_module files
    • $ exit                                    # Leave the Docker container
$ make generate-config file={FILENAME}    # Generate files from configuration files through docker container

  5. Build FreeTACOS project

    • $ make enter                              # Enter Docker container
$ cd src/kernel                           # Enter kernel folder
$ make                                    # Build FreeTACOS project

  6. Deploy to SD-card (First mount SD-card)

    • $ cd aarch/circle-stdlib/libs/circle      # Enter circle
$ ./makeall clean                         # Clean build filed
$ ./makeall                               # Build files
$ cd boot                                 # Go to boot folder
$ make bootloader64                       # Build bootloader files
Copy generated files from boot to SD-card

    • $ exit                                    # Exit Docker container
$ cd /app/src/kernel                      # Change directory to FreeTACOS kernel
Copy kernel8-rpi4.elf, kernel8-rpi4.img, kernel8-rpi4.lst and kernel8-rpi4.map to SD-card
  7. Insert card and Power Raspberry Pi 4
